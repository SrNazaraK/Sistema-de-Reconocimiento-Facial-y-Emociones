# Sistema de Reconocimiento Facial y Análisis de Emociones

Sistema modular desarrollado en Python para el registro de personas, identificación en tiempo real y análisis de las 7 emociones básicas.

## Requisitos
- Python 3.9+
- OpenCV
- DeepFace
- SQLite

## Estructura
- `src/core`: Lógica de IA.
- `src/gui`: Interfaz gráfica.
- `src/database`: Persistencia de datos.
